# 🚗 2D Car Racing Game

A simple and fun **2D car racing game** built using HTML, CSS, and JavaScript. The player controls a car, avoids enemy vehicles, and scores points while surviving as long as possible.

---

## 🎮 Game Overview

This game is a browser-based car racing simulation where:

* You control a car using keyboard arrows
* Avoid collisions with enemy cars
* Score increases over time
* You have limited lives ❤️
* Game ends when all lives are lost

---

## ✨ Features

🚗 **Player Car Control**

* Move left and right using arrow keys

🚙 **Enemy Cars**

* Randomly generated enemy vehicles
* Increasing challenge with continuous gameplay

🛣️ **Animated Road**

* Moving lane lines create a realistic driving effect

🎯 **Score System**

* Real-time score tracking
* Best score saved during gameplay

❤️ **Lives System**

* Player gets 3 lives before game over

⏸️ **Pause / Resume**

* Pause and resume the game anytime

🔊 **Sound Effects**

* Crash sound on collision
* Point sound when passing enemies

🔁 **Restart Option**

* Restart the game after game over

---

## 🛠️ Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling and animations
* **JavaScript (Vanilla JS)** – Game logic and interactivity

## 🎮 How to Play

1. Open the game in your browser
2. Use controls:

| Key            | Action     |
| -------------- | ---------- |
| ⬅️ Left Arrow  | Move Left  |
| ➡️ Right Arrow | Move Right |

3. Avoid enemy cars
4. Score points and try to beat your best score

---

## 🚀 How to Run Locally

1. Download or clone the repository:

```bash
git clone https://github.com/your-username/2d-car-game.git
```

2. Open the folder
3. Run the game:

* Double-click `index.html`
* Or open it in any browser
